1. Bill of Materials

3D Printed Parts

The hand itself is based on an existing open-source model. Grab the STL files for the fingers, palm, and wrist mounting directly from Thingiverse:

  • Model: InMoov Hand / Robotic Hand
  • Creator: Ryan Gross (grossrc)

Electronics & Hardware

  • ESP32 Microcontroller (e.g., ESP32 WROOM-32 DevKit)
  • PCA9685 16-channel PWM/servo driver (I2C)
  • 4x Servo Motors (MG996R or SG90)
  • Webcam (Built-in or USB)
  • Power Supply: External 5V - 6V capable of handling at least 3 amps.
  • Tendons: Fishing line or braided thread, and elastic cord for the return tension.
2. Hardware Assembly

The Physical Build

Before touching the electronics, put the 3D-printed hand together. This YouTube assembly video shows exactly how to string the tendons and place the servos.

Wiring the Electronics

The PCA9685 talks to the ESP32 over I2C. Connect them as follows:

PCA9685 PinESP32 Pin
VCC3.3V
GNDGND
SDAGPIO 21
SCLGPIO 22

Plug the four servos into the PCA9685: Ch 0 (Thumb), Ch 1 (Index), Ch 2 (Middle), and Ch 3 (Ring + Pinky shared).

⚠️ Do not power the servos directly from the ESP32's 5V pin. Connect your external 5V/6V power supply directly to the V+ and GND screw terminals on the PCA9685. Tie the ESP32 GND and the PCA9685 GND together.

3. Software Setup

Flashing the ESP32

  1. Install the Arduino IDE.
  2. Add ESP32 boards to the Boards Manager.
  3. Install the Adafruit PWM Servo Driver Library.
  4. Open the .ino file in the esp32/ folder of the repo.
  5. Plug in your ESP32, pick the right COM port, and upload.

Running the Python Tracker

The Python script uses MediaPipe to track your hand and sends serial commands to the ESP32.

python -m venv venv source venv/bin/activate pip install opencv-python mediapipe pyserial

Update the script with your ESP32's COM port, and run it:

python main.py

Hold your hand up to the webcam, and watch the robot mirror your movements!
